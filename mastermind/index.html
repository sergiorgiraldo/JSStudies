Mastermind Number Game

How to Play:

Try to guess the secret 5-digit code. Each digit can be from 0-9.

After each guess, you'll get feedback:

  • Black background: Correct number in the correct position
  • White background: Correct number but in the wrong position
  • Gray background: Number not in the code

You have 8 attempts to crack the code!
