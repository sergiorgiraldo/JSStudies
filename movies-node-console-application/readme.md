# movies CLI

A CLI for movies.

## done with glue gun

```shell
$ npx gluegun new movies
$ cd movies
... do your thing
$ yarn link
$ cd bin
./movies ...
```

## Customizing your CLI

Check out the documentation at https://github.com/infinitered/gluegun/tree/master/docs.

## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami
$ npm test

$ npm publish
```
